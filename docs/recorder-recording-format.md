# Recording format (v1)

Recorder persists capture data in **IndexedDB** while recording and, when you click **Export**, downloads a single zip into Chrome’s **Downloads** folder.

There is **no** session-level `metadata.json` at the zip root for the primary workflow—value lives in **per-page `.txt`** files.

## Downloaded zip filename

The archive name is:

`recorder-session-YYYY-MM-DDTHH-mm-ss.zip`

The timestamp uses **UTC** fields (`getUTC*`) so the basename is stable regardless of the machine timezone. Time segments use **hyphens** instead of colons so the name is filesystem-safe.

Example: `recorder-session-2026-05-05T14-30-00.zip`.

## Inner layout

One folder per **site bucket** (hostname with `.` → `-`), each containing one `.txt` per captured URL slug:

```text
recorder-session-2026-05-05T14-30-00.zip
├── www-example-com/
│   └── www-example-com-docs-page.txt
└── app-example-com/
    └── app-example-com-dashboard-index.txt
```

Each `.txt` concatenates **snapshot blocks** for that URL in append order. Adjacent blocks are separated by a line containing only:

```text
----------
```

## Snapshot block contents

Each block is generated by the default **tree-outline** pipeline: a **page metadata** section (title, URL, capture time, head-derived fields where available) plus **page content** as a readable tree-style outline of the page (via `html-textify` + `buildSnapshotBlockText` under `extensions/recorder/src/`).

v1 does **not** include site-specific parsers in the registry; only the default outline is used.

## IndexedDB model (summary)

- **Store 1 — `polled_unique`:** dedupe key = SHA-256 hex of full `outerHTML`; holds `rawHtml` and metadata for the worker.
- **Store 2 — `processed_by_url`:** `fullUrl` → `snapshots: { id, text }[]`.
- **Store 3 — `snapshot_ledger`:** monotonic `seq` for global **oldest-first** deletion (trim old).

The in-memory **digest queue** is not persisted; stopping recording clears the queue and store 1.

## Tests and fixtures

Unit tests live under **`extensions/recorder/tests/`** (schema, memory IDB shape, pipeline, export zip bytes/naming).
